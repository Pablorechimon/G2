## **Instalacion**
1- [Instalar Python 3.10](https://www.python.org/downloads/release/python-3104/)

2- Instalar pipenv con
```
pip install pipenv
```

3- Instalar dependencias con
```
pipenv install
```
4- Instalar el servicio
